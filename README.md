## 🛠 掲載アプリ一覧

### 1. 🔥 激怒クレーム対応シミュレーター
新人スタッフが、理不尽なクレームに対する「一次対応」を安全に練習できるロールプレイングアプリです。

<img src="[ここにアプリのスクショ画像のURL]" width="600">

#### 課題背景
店長時代、クレーム対応のストレスで新人が早期離職してしまう課題がありました。しかし、対人でのロールプレイング研修は店長の時間を圧迫するため、十分な回数をこなせないジレンマがありました。

#### 解決策
AIを「理不尽な客」になりきらせることで、店長の時間を奪わずに、スタッフが何度でも失敗できる練習環境を構築しました。

---

### 2. 業務マニュアル検索AI（RAG）
分厚い業務マニュアルの中から、必要な情報を即座に検索・回答してくれるAIチャットボットです。

#### 課題背景
複雑な新メニューやコラボメニュー等ルールが多い商品を販売する際、現場が混乱していることがよくありました。紙のマニュアルは探すのに時間がかかり、実用的ではありませんでした。

#### 解決策
店舗のマニュアル（今回は架空のデータを使用）をAIに学習させ、自然言語で質問するだけで、該当箇所を引用して即答する仕組みを構築しました。

#### 使用技術
*   **LangChain / FAISS:** ベクトル検索による高速な情報抽出
*   **RAG (Retrieval-Augmented Generation):** ハルシネーションを抑制し、マニュアルに基づいた回答のみを生成
